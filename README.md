# Web-based documents template

This repo contain source-code for the website-based documents template for any project that need to have a website-based documents.

This website target to build a simple and easy to config website-based documents for any project. It's responsive and support markdown.

## Features
- Simple and easy to config
- Responsive design
- Support markdown

## Tech stack
- HTML
- CSS
- JavaScript

- Lit (Web Components)
- DaisyUI (Tailwind CSS Components)
- Tailwind CSS
